# 🚛 DriveSafe Fleet  
**Segurança Inteligente para Frotas com IA**

Sistema inovador para monitoramento de **fadiga e consumo de álcool** em motoristas, utilizando inteligência artificial e sensores inteligentes para prevenir acidentes e salvar vidas.

---

## 📌 Visão Geral

O **DriveSafe Fleet** é uma solução tecnológica voltada para empresas de transporte, frotas corporativas e motoristas de aplicativo.  

O sistema realiza **monitoramento em tempo real** do estado do condutor, detectando sinais de:

- 🚫 Consumo de álcool (detecção passiva, sem sopro)
- 😴 Fadiga e sonolência
- ⚠️ Comportamentos de risco

---

## 🎯 Objetivos

- Reduzir acidentes causados por álcool e fadiga  
- Proteger motoristas e cargas  
- Aumentar a eficiência operacional  
- Fornecer dados para tomada de decisão  

---

## ⚙️ Funcionalidades

### 🧠 Detecção de Fadiga
- Análise facial em tempo real  
- Monitoramento de piscadas  
- Identificação de sonolência  

### 🍺 Detecção de Álcool
- Sensores integrados ao veículo  
- Detecção automática e passiva  
- Alertas imediatos  

### 📊 Dashboard
- Dados em tempo real  
- Histórico de motoristas  
- Relatórios e alertas  

### 📍 Monitoramento
- Localização da frota  
- Status de segurança por veículo  

### 🚨 Alertas
- Alertas visuais e sonoros  
- Notificações para gestores
